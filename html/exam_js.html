Diferencias entre etiqueta <p> y etiqueta <div>

La etiqueta <p> tiene un margen y formato predefinido mientras que la etiqueta <div> no tiene formato

¿Qué es minificación de CSS?

Sirve para comprimir un archivo CSS. Elimina espacios, agrupa redundancias y quita saltos de linea

¿Por qué no usar WYSIWYG o CMS?

Es poco profesional, tiene limitaciones como inclusión de JavaScript y el código es menos limpio.

¿cómo poner variables en CSS?

:root { --color_principal : green; }

¿Qué es un preprocesador de CSS?

Le da super poderes al CSS (extiende las funcionalidades de un CSS), permitiéndonos tener variables, funciones, mixins, reutilizar código y tener más flexibilidad al momento del desarrollo.

Diferencia entre innerHTML y write

Después de innerHTML va un igual para asignarle un valor. En write lleva el valor entre paréntesis

¿Qué es una librería?

Conjunto de instrucciones definidas para un fin o funcionalidad. Se puede usar u omitir sin efectos secundarios.

En JavaScript ¿quée es getElementById?

Se usa para obtener un elemento por su identificador (para decirle ¡hey tú! el que tiene este Id)

Menciona 3 eventos para ejecutar una función en JavaScript
  • onclick: cuando das click.
  • onload: cuando la página ha sido cargada
  • onkeypress: cuando presionas una tecla.
Diferencia entre JavaScript y Java

Java es un lenguaje compilado multipropósito y JavaScript es un lenguaje interpretado para dar interactividad a páginas web.
